[HTML] HTML? 설명해보자

이번 문서의 목적 HTML의 정의를 설명할 수 있다. HTML의 기본 구조를 설명할 수 있다. HTML Element가 무엇인지, 구성요소는 무엇인지 설명할 수 있다. HTML? HTML은 Hyper Text Mark up Language의 약자입니다. 우리가 흔히 웹 페이지! 라고 부르는 것들이 사실 HTML문서입니다. Hyper Text는 우리가 흔히 ‘링크’라고 부르는 것이라고 일단 알겠는데, Mark up Language는 뭘까요? Markup Language? Markup Language는 어떤 언어일까요? Markup Language는 간단히 말해 ‘문서의 표현형식, 논리적 구조를 설명하기 위한 언어’입니다. 강조할 때는 이것처럼 굵게 쓰는 것, 제목과 본문처럼 구조를 나타내는 문법을 가진 것이 특징입니다....
